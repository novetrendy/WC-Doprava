# WC-Doprava
Implementace českých kurýrních služeb (DPD Pickup, Česká pošta, WE|DO, GLS) do Woocommerce

Pro správnou funkčnost pluginu je nutné mít volbu "Další upřesnění k adrese pole" nastavenou na Volitelné / Povinné.
